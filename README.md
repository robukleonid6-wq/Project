# Простой ВЕБ сервер на Python

Реализация веб-сервера на Python с использованием сокетов.

## Возможности

- Обработка HTTP GET и POST запросов
- Обслуживание статических файлов (HTML, CSS, JS, изображения)
- Поддержка различных MIME-типов
- Обработка ошибок (404, 500)
- Простой API
- Поддержка форм
- Многопоточная обработка запросов

## Структура проекта
├── server.py # Основной файл сервера
├── requirements.txt # Зависимости проекта(в данном проекте они отсутствуют)
├── README.md # Документация
└── static/ # Директория со статическими файлами
├── index.html # Главная страница
├── form_success.html # Форма успешного отправления
├── form.html # Форма
├── error.html # Траница ошибки
├── skyrim.html # Страница игры
├── welcome.html # страниза 
├── style.css # Страница приветствия
├── script.js # JavaScript
└── images/ # Изображения
